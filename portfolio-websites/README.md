## 🎨 Design Features

- **Modern Clean Design** - Desain bersih dan profesional

## 🚀 Technology Stack

- **Frontend**: Next.js 15, React 19, TypeScript
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Forms**: React Hook Form
- **Animation**: Framer Motion
- **SEO**: Next.js SEO
